# LinkedIn and Resume Content

## LinkedIn Project Title

U.S. Grocery Demand & Inventory Intelligence Pipeline | AWS + Databricks + PySpark

## LinkedIn Description

Built an end-to-end U.S. grocery demand and inventory intelligence pipeline using AWS Lambda, Amazon S3, Databricks, PySpark, Spark SQL, Delta Lake, and Databricks AI Functions.

The pipeline ingests real Instacart market basket data from Kaggle into S3 using AWS Lambda, then processes 33M+ order-product records through Bronze, Silver, and Gold lakehouse layers. Developed curated analytics tables for product demand trends, reorder behavior, basket affinity, derived inventory snapshots, stockout risk, and overstock exposure.

Implemented Databricks AI enrichment to classify grocery products into business-friendly product themes and demand types, using a hybrid AI + fallback approach to balance accuracy, runtime, and cost.
